## Project Description
This repository contains automated functional and UI test cases for the **Swift Translator** (Singlish to Sinhala conversion tool). The purpose of this assignment is to evaluate the accuracy, robustness, and user interface of the translator using Playwright automation.

## Test Suite Overview
The test suite covers a total of 35 test cases:
- **Positive Functional Tests (24):** Validating correct translations for daily usage, grammar, and technical terms.
- **Negative Functional Tests (9):** Testing the system's robustness against spacing issues, mixed English, and formatting errors.
- **UI/Real-time Tests (2):** Ensuring the output updates correctly in real-time as the user types.

## Tools & Technologies
- **Automation Framework:** Playwright
- **Language:** JavaScript
- **Version Control:** Git & GitHub
- **Environment:** Node.js
